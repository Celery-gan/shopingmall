1.拿到server文件夹，先安装依赖，安装依赖成功之后，在浏览器访问localhost:7001，显示导入数据成功，检查数据库中是否有shoppingMall这个数据库，
  如果有，则说明导入数据成功，成功之后，只需要把服务打开，其余的不需要管
2.所有测试的接口根路径为localhost:7001/v1/xxx，接口测试一律在postman当中进行
3.前端项目请求接口的根路径为 http://localhost:7001/v1
4.接口详情在Api.js这个文件当中
5.所有用到的第三方插件和库
  1.better-scroll(页面平滑滚动)
  2.fastclick(取消移动端点击延迟)
  3.vant(ui组件库)
  4.移动端适配方案 (vw适配)
    安装postcss-aspect-ratio-mini postcss-cssnext postcss-import postcss-px-to-viewport postcss-url postcss-viewport-units postcss-write-svg
    然后在package.json添加以下代码:
	"postcss": {
    "plugins": {
      "postcss-import": {},
      "postcss-url": {},
      "postcss-aspect-ratio-mini": {},
      "postcss-write-svg": {
        "utf8": false
      },
      "postcss-cssnext": {},
      "postcss-px-to-viewport": {
        "viewportWidth": 375,
        "viewportHeight": 667,
        "unitPrecision": 3,
        "viewportUnit": "vw",
        "selectorBlackList": [
          ".ignore",
          ".hairlines"
        ],
        "minPixelValue": 1,
        "mediaQuery": false
      },
      "cssnano": {
        "cssnano-preset-advanced": {
          "preset": "advanced",
          "autoprefixer": false,
          "postcss-zindex": false,
          "zindex": false
        }
      }
    }
  }
